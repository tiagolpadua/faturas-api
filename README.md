## API
`http://localhost:3200/`  

Response formats: JSON

## Endpoints
### Users `/users`:
**Accepts:**  
    - `GET`  
    - `POST`  
```
{
    name: '',
    username: '',
    phone: '',
    password: '',
    avatar: ''
}
```

#### Parameters 
##### ` users/validation/:username`
**Accepts:**  
    - `GET`  
  
##### `user/:userId`
**Accepts:**  
    - `GET`  
    - `PATCH`   
    
### Login `/login`:
**Accepts:**  
    - `POST`
```
{
    email: '',
    password: ''
}
```

### Email `/emails`:
Headers: `authorization`

**Accepts:**  
    - `POST`  
    - `GET`

```
{
    to: '',
    subject: '',
    content: ''
}
```

#### Parameters `emails/:emailId`:
**Accepts:**  
    - `DELETE`
